# How do I get started? 
1. **Wordpress**
- Go to your raspberrypi server
- start apache server
- go to http://raspberrypi/

2. **Mail Service**
- Go to your raspberrypi server
- php ./sender.php
- When executed,
* insert Recipient email address
* insert mail title
* insert mail contents
- Then, you can send the mail
